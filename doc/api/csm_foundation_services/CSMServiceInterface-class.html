CSMServiceInterface

CSMServiceInterface class abstract interface

Interface for CSMService.

Defines require behavior for a CSMService implementation.

CSMService concept: is a set of operations where get data by protocol transactions, like an endpoint. (i.e.) google.com/security <-- service; google.com/security/calculate <-- act, where the operation will be solved with data.
